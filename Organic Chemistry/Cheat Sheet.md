## Significant figures

Number of digits that can be justified by the data.
> The number of significant figures (sf) in the result of a calculation cannot exceed the number of significant figures in the data!

| Value                | # of sf      | Explanation                                                             |
| -------------------- | ------------ | ----------------------------------------------------------------------- |
| 5                    | 1 sf         |                                                                         |
| 69                   | 2 sf         |                                                                         |
| 1.5                  | 2 sf         |                                                                         |
| 10.5                 | 3 sf         | The zero is actually measured                                           |
| 10.05                | 4 sf         | The zeroes are actually measured                                        |
| 10.50                | 4 sf         | The zeroes are actually measured                                        |
| 0.5                  | 1 sf         | The leading zero is a placeholder for a power of 10                     |
| $2.5 \times 10^{-3}$ | 2 sf         | The powers of 10 only show the magnitude, the actual digits are 2 and 5 |
| 40                   | 1 sf or 2 sf | Ambiguous, use scientific notation                                      |
| 40.                  | 2 sf         | Unambiguous, the trailing decimal point shows that the zero is measured |
| 40.0                 | 3 sf         | Both zeroes are measured                                                |

**Precision** - how close measurements are to one another; reflects the # of sf.
**Accuracy** - how close measurements are to the "true value".

1. **Rounding**:
	- last digit > 5 → up 
	- last digit < 5 → down
	- last digit = 5 → to the nearest even number
2. **Addition & subtraction**:
	Result must have # of sf equal to the *smallest # of decimal places* in the data
	$0.10 \space g + 0.024 \space g = 0.12 \space g$
3. **Multiplication & division**:
	Result must have # of sf equal to the *smallest # of sf* in the data
	$\frac{8.62 \space g}{2.0 \space cm^3} = 4.3 \space g \cdot cm^{-3}$
4. **Integers & exact numbers**:
	Uncertainty of the result is determined by the measured value
	Exact numbers are treated as if they have an infinite # of sf
5. **Logarithms & exponentials**:
	The mantissa (decimal part) of a common logarithm ($log_{10}$) has the same # of sf as the original number and vice versa.
	$log(2.45) = 0.389, 10^{0.389} = 2.45$
	No simple rule for natural logarithms.
