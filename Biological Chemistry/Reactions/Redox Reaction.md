One element oxidizes, another reduces.

**Oxidation** - loss of electrons.
**Reduction** - gain of electrons.

Redox reactions are numerous and appear in all kinds of circumstances.

Oxidation of magnesium:
$\ce{2 Mg (s) + O2 (g) -> 2 MgO (s)}$
$\ce{Mg^0 - 2 e- -> Mg^{2+}} \text{ - magnesium oxidizes}$
$\ce{O2^0 + 4 e- -> 2 O^{2-}} \text{ - oxygen reduces}$

Redox can also be a substitution reaction:
$\ce{2 NaBr (s) + Cl2 (g) -> 2 NaCl (s) + Br2 (l)}$
$\ce{2 Br^{-1} + 2 e- -> Br2^{0}} \text{ - bromine oxidizes}$
$\ce{Cl2^0 - 2 e- -> 2 Cl-} \text{ - chlorine reduces}$

## Oxidation number

$N_{ox}$ - oxidation increases, reduction decreases.

In monoatomic ions, oxidation number = charge

## Agents

**Oxidizing agent (oxidant)** - oxidizes *another* reagent.
**Reducing agent (reductant)** - reduces *another* reagent.

|                      | Oxidant                 | Reductant            |
| -------------------- | ----------------------- | -------------------- |
| **Behavior**         | causes oxidation        | causes reduction     |
| **Electrons**        | gained                  | lost                 |
| **Oxidation number** | decreases               | increases            |
| **Typical example**  | $\ce{O2, \space H2SO4}$ | $\ce{H2, \space CO}$ |

$\ce{2 Mg (s) + O2 (g) -> 2 MgO (s)}$
$\ce{Mg} \text{ oxidizes itself, reduces } \ce{O2} \text{ - reductant}$
$\ce{O2} \text{ reduces itself, oxidizes } \ce{Mg} \text{ - oxidant}$

$\ce{Zn^0 (s) + Cu^{2+} (aq) -> Zn^{2+} (aq) + Cu^0 (s)}$
$\ce{Zn} \text{ oxidizes itself, reduces } \ce{Cu} \text{ - reductant}$
$\ce{Cu} \text{ reduces itself, oxidizes } \ce{Zn} \text{ - oxidant}$

#### Metal Extraction

Getting pure iron from iron(III) oxide:
$\ce{Fe2O3 (s) + 3 CO (g) -> 2 Fe (l) + 3 CO2 (g)}$

#### Combustion

Burning fuel (heptane):
$\ce{C7H16 (g) + 11 O2 -> 7 CO2 (g) + 8 H2O (g)}$
